# mini-search-engine
Mini Search Engine - Advanced Programming Techniques Course Project - Cairo Uni.

# Install Dependencies

* maven +3.6.0
* java +11.0.3
* javac +11.0.3
* nodejs +8.11.4
* yarn +1.21.1

```
$ mvn clean install
$ yarn install
```

# Build Web

`$ yarn build`

# Run Server

`$ mvn spring-boot:run -Dserver.port=<PORT>` where `<PORT>` is the port to bind to.

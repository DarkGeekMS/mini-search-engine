# mini-search-engine
Mini Search Engine - Advanced Programming Techniques Course Project - Cairo Uni.

# Install Dependencies

* maven +3.6.0
* java +11.0.3
* javac +11.0.3

# Run Server

in server dir: `$ mvn spring-boot:run -Dserver.port=<PORT>` where `<PORT>` is the port to bind to.
